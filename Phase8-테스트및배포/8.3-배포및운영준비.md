## 배포 전략
```yaml
# Blue-Green 배포 예시
version: '3.8'
services:
  sbs-batch-blue:
    image: sbs-batch:latest
    environment:
      - SPRING_PROFILES_ACTIVE=prod

  sbs-batch-green:
    image: sbs-batch:new
    environment:
      - SPRING_PROFILES_ACTIVE=prod
```

## 모니터링 설정

- 로그 수집 시스템

- 네트워크 타임아웃 설정
