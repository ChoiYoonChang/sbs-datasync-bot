## 크론 표현식 도우미
```typescript
interface CronExpression {
  second: string;
  minute: string;
  hour: string;
  dayOfMonth: string;
  month: string;
  dayOfWeek: string;
  year?: string;
}

interface ScheduleTemplate {
  name: string;
  description: string;
  cronExpression: string;
  category: 'daily' | 'weekly' | 'monthly' | 'custom';
}
```
