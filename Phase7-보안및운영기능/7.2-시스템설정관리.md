## 설정 카테고리
```java
public enum ConfigCategory {
    DATABASE,       // DB 연결 설정
    BATCH,         // 배치 실행 설정
    NOTIFICATION,  // 알림 설정
    SECURITY,      // 보안 설정
    PERFORMANCE,   // 성능 설정
    LOGGING        // 로깅 설정
}
```
